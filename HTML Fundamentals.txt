 html (hypertext markup language) is the standard language used to create webpages. it structures the content and content tells the browser how to display text, image, links, and other elements

 <!DOCTYPE html>
 <html>
  <head>
   <title>page title </title>
  </head>
 <body>
  <h1> this is a heading</h1>
  <P> this is a paragraph </p>
 </body>
</html>


 <!DOCTYPE html> : declares the document type
 <html> : root elemment of the html document 
<head> : contains metadata (e.g.,  title , stylrs, scripts)
<title> : title of the document shown in the browser tab)
 <body> : contains the visible content
